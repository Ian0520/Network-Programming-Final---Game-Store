# Developer Game Template

This folder contains a minimal game package skeleton used by
`hw3/developer/create_game_template.py`.

Placeholders:
- `__GAME_ID__` will be replaced with the requested gameId.

Files:
- `manifest.json`: required metadata + entrypoints
- `server_main.py`: minimal game server example
- `client_main.py`: minimal game client example
- `framing.py`: length-prefixed JSON framing helper (self-contained)
